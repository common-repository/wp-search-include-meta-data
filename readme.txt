WordPress plugin to include post meta data when using the default search

== Description ==

This WordPress Plugin makes the default search also search based on meta data.

* No setup required!

== Installation ==

Upload the "wp-search-include-meta-data" folder to your wp-content/plugins directory, then Activate it.
